Getting Started

Creating a Microsoft PowerPoint presentation is different from creating a Microsoft Word document. You have to first create a pptx object and then to add a slide before to be able to send R outputs into that document (into that slide...). Finally, write the object into a file *.pptx.

Below a commented R script: 

library( ReporteRs )

# Here we define powerpoint document filename to write
pptx.file = "presentation.pptx"

# Creation of doc, a pptx object (default template)
doc = pptx( )

# check my layout names:
slide.layouts(doc)


doc = addSlide( doc, "Two Content" )
# add into doc first 10 lines of iris
doc = addTitle( doc, "First 10 lines of iris" )
doc = addTable( doc, iris[1:10,] )

# add text with stylename "Normal" into doc (and an empty line just before)
doc = addParagraph( doc, value = c("", "Hello World!"), stylename = "Normal" )

doc = addSlide( doc, "Title and Content" )
# add a plot into doc 
doc = addPlot( doc
                , function() plot( rnorm(10), rnorm(10) )
)

# write the doc 
writeDoc( doc, pptx.file )

When creating a pptx object, a template is used. This file is the original pptx document that will be completed with R ouputs. Formats, styles and slide layouts will be those available in the template file.

Slide Layouts

Available layouts are thoses contained in the template. Slide layout concept is important in ReporteRs.

Content is conditionned by the choosen layout. When a slide is added with layout 'Title and Content', this slide will only be able to contain 2 contents: a title and a content theses are shapes.

Shapes are where most of the formatting is done: plot size is defined by shapes dimensions in the layout, default fonts and paragraphs styles are thoses defined by the shape properties, etc. 

doc = pptx()
slide.layouts(doc)
 [1] "Title Slide"             "Title and Vertical Text"
 [3] "Title and Content"       "Two Content"            
 [5] "Section Header"          "Vertical Title and Text"
 [7] "Content with Caption"    "Title Only"             
 [9] "Comparison"              "Blank"

slide.layouts returns slide layouts names. When using adding a slide with addSlide you may need that function to ckeck available names.

Plot design

use optional argument layout with function slide.layouts to get a plot of the design of your layout 

doc = pptx()
slide.layouts(doc, "Comparison" )

Here, the slide can receive the following shapes: a title, 4 contents (plots, tables, paragraphs), a date and a slide number.

Available Shapes

Titles

Function addTitle add a title in the current slide. It has no level concept.

A title is a specific shape that is (most of the time) present in a slide or slide layout. 

# Create a new document
pptx_doc = pptx( title = "title" )

# add a slide with layout "Title and Content"
pptx_doc = addSlide( pptx_doc, slide.layout = "Title and Content" )

# Here we fill the title shape with "My title"
pptx_doc = addTitle( pptx_doc, "My title" )

# Write the object in file "~/presentation.pptx"
writeDoc( pptx_doc, "~/presentation.pptx" )

Contents

See functions addTable, addFlexTable, addPlot and addParagraph for details.

Date, page number and footer

  • Function addDate add a date shape in the current slide.
  • Function addFooter add a footer shape in the current slide.
  • Function addPageNumber add a page number shape in the current slide.

Again, this is only working if the layout used to add a new slide is containing the corresponding shapes. 

# Create a new document
pptx_doc = pptx( title = "title" )
# add a slide with layout "Title Slide"
pptx_doc = addSlide( pptx_doc, slide.layout = "Title Slide" )
pptx_doc = addTitle( pptx_doc, "Presentation title" ) #set the main title
pptx_doc = addSubtitle( pptx_doc , "This document is generated with ReporteRs.")#set the sub-title

## add a page number on the current slide
pptx_doc = addPageNumber( pptx_doc )

pptx_doc = addSlide( pptx_doc, slide.layout = "Title and Content" )
## add a page number on the current slide but not the default text (slide number)
pptx_doc = addPageNumber( pptx_doc, value = "Page number text")

# Write the object in file "~/presentation.pptx"
writeDoc( pptx_doc, "~/presentation.pptx" )

Replace/Insert content

To replace an existing slide by a new one, use addSlide with optional argument bookmark. Bookmark is the index of the slide that has to be replaced with a new empty one. 

# Create a new document
doc = pptx( title = "title", template = "MyFileThatNeedAGraphToBeUpdated.pptx" )

# replace slide 3 of MyExistingFileThatNeedAGraphToBeUpdated.pptx 
# by a new slide with layout "Title and Content" then add content
doc = addSlide( doc, slide.layout = "Title and Content", bookmark=3 )
doc = addTitle( doc, "my new graph")
myplot = qplot(Sepal.Length, Petal.Length, data = iris, color = Species, size = Petal.Width, alpha = I(0.7))
doc = addPlot( doc, print, x = myplot )

# Write the object in file "~/presentation.pptx"
writeDoc( doc, "MyFileWithAGraphUpdated.pptx" )
