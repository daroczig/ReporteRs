Getting Started

To create a WEB presentation, you have to first create an html object and then to add a page before to be able to send R outputs into that document (into that page...). Finally, write the object (into a directory).

Below a commented R script: 

html.directory <- "HTML"

doc = html( title = "My document" )

doc = addPage( doc, "Demo" )
doc = addTitle( doc , "First 5 lines of iris", level = 1)
doc = addTable( doc , iris[1:5, ])

doc = addTitle( doc , "ggplot2 example", level = 1)
myggplot = qplot(Sepal.Length, Petal.Length, data = iris, color = Species, size = Petal.Width )
doc = addPlot( doc = doc , fun = print, x = myggplot )

doc = addTitle( doc , "Text example", level = 1)
doc = addParagraph( doc, "My tailor is rich.", stylename = "text-primary" )

xx = writeDoc( doc, html.directory )
browseURL( xx )

html objects are created with function html, there is no template argument yet.

Page Layouts

The layout is the same than the one used with the page you are actually reading (generated with ReporteRs).

  • pages links will be displayed in the top menu bar
  • pages TOC will automatically be displayed in the left part of web pages.
  • Content will be displayed on the right of the TOC.

Available content elements

Titles

Function addTitle add a title in the current slide. 

# Create a new document
html_doc = html( title = "title" )

# add a page with title "My page title"
html_doc = addSlide( html_doc, title = "My page title" )

# Here we add a first level title
html_doc = addTitle( html_doc, "My title", level = 1)

# Write the object in a directory
writeDoc( html_doc, "html_doc" )

Contents

See functions addTable, addFlexTable, addPlot and addParagraph for details.

R Script

Use addRScript to add a script in a web document.
