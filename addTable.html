Content details (See section Usage)

  • Headers are columns names, labels can be modified.
  • Grouped header is a row preceding headers where column labels can be gathered under a label. This row is optional.
  • Each column has one of the following reporting types : double, integer, character, percent, date, datetime, logical.
  • Data will be formatted based on the reporting types of each columns, for example, all numeric columns will be formated the same.

Formatting tables (See section Formatting table layout)

  • Elements that can be formated are cells, paragraphs and texts.
  • Other options are existing:.
    • the number of digits to the left of the decimal point for numeric data
    • the number of digits to the left of the decimal point for percent data
    • percent symbol to add after data of reporting type 'percent'

  • doc, the doc object where to add the table

Coloring data background cells

Argument columns.bg.colors specifies background colors of the table content.

It affect table body content not the whole table (header content will not be affected)

Background colors are specified by columns, columns.bg.colors is a named list, names are data column names and values are character vectors specifying background colors ; each element of the list is a vector of length nrow(data).

This argument is optional.

The following code show usage of this parameter: 

# dummy data.frame
data = data.frame(
  colA = 1:5
  , colB = c("A", "B", "C", "D", "E")
  , colC = c(TRUE, FALSE, TRUE, FALSE, TRUE)
  )

doc = addTable( doc
  , data
  , columns.bg.colors= list(
      # spec for colA background colors
      colA = c("#527578", "#84978F", "#ADA692", "#47423F", "#B3B1B2")
      # spec for colC background colors
      , colC = c("#74A6BD", "#7195A3", "#D4E7ED", "#EB8540", "#B06A3B")
      # leave colB background colors unspecified
  )
)

Coloring data font

Argument columns.font.colors specifies font colors of the table content.

It affect table body content not the whole table (header content will not be affected)

Font colors are specified by columns, columns.font.colors is a named list, names are data column names and values are character vectors specifying font colors ; each element of the list is a vector of length nrow(data).

Formatting table layout

Table formatting properties are defined with an object of class tableProperties.

Headers, grouped headers and content can be formatted at different levels( cell properties, paragraph properties, text properties).

Headers share all the same format.

Grouped headers share all the same format.

Content is formatted according to its data type (argument col.types in addTable). double columns share all the same format, character columns share all the same format, etc.

Conditionnal formating is not specified in tableProperties but in addTable. See Coloring data background cells and Coloring data font.

Below an example to understand how it works 

data = data.frame(
    col1 = c("a", "b", "c", "d" )
  , col2 = 1:4
  , col3 = rnorm(4)
  , col4 = Sys.Date() + 1:4
  , col5 = (1:4)/4
  , col6 = c(T,F,T,F)
)

doc = addTable( doc, data
, header.labels = c("Header 1", "Header 2"
    , "Header ...", "Header ...", "Header ..."
    , "Header <code>p</code>"
  )
, groupedheader.row = list( values = c("grouped header 1", "", "grouped header k"), colspan = c(2, 1, 3) )
, layout.properties = tableProperties( groupedheader.par = parProperties( text.align = "right" )
    , header.par = parProperties( text.align = "right" )
    , data.par = parProperties( text.align = "right" )
    , character.cell = cellProperties( background.color = "#00ACE9")
    , integer.cell = cellProperties( background.color = "#D43F3F")
    , double.cell = cellProperties( background.color = "#6A9A1F")
    , percent.cell = cellProperties( background.color = "#893667")
    , date.cell = cellProperties( background.color = "#404040")
    )
)

Formatting table headers

Headers

To format table headers, use the following arguments:

  • header.text: a textProperties object ; text formatting properties of column headers.
  • header.par: a parProperties object ; paragraph formatting properties of column headers.
  • header.cell: a cellProperties object ; cell formatting properties of column headers. 
# let's format header labels with a bold text inside a centered paragraph
myformat = tableProperties(
  header.text = textProperties(font.weight="bold", font.size = 15)
  , header.par = parProperties( text.align = "center" )
)
doc = addTable( doc, iris[1:5,], layout.properties= myformat )

doc = addParagraph( doc, c( "" ), stylename = "normal" )

# many possible options
myformat = tableProperties(
  header.text = textProperties(
      font.weight = "bold"#normal
      , color = "#DCDDD8"
      , font.size = 15
      , font.style = "normal"#italic
      , underlined = FALSE
      , font.family = "Arial"
      )
  , header.par = parProperties( text.align = "center", padding=5 )
  , header.cell = cellProperties(
        background.color="#475F77"
        , border.color="#354B5E"
        , border.bottom.style="dashed"
        )
)
doc = addTable( doc, iris[1:5,], layout.properties = myformat )

Formatting table content

Each column can have one of the following reporting types : double, integer, character, percent, date, datetime.

For each type, there are 3 arguments to specify column formattting ( cell properties, paragraph properties, text properties).

Columns with identical types share all the same format.

Everything is specified in the argument layout.properties with a tableProperties object (returned by usage of function tableProperties). 

?addTable
?tableProperties

tableProperties object is a container for table formatting options.

Shorcuts

3 shorcuts can be used to set the identical format for any type of data: data.cell, data.par, data.text

data.text : a textProperties object ; text formatting properties for columns of any type.

data.par : a parProperties object ; paragraph formatting properties for columns of any type.

data.cell : a cellProperties object ; cell formatting properties for columns of of any type. 

doc = addTable( doc
        , data = data.frame( col1 = 1:5, col2 = rnorm(5), col3 = letters[1:5]
                        , col4 = as.Date("2011-09-04", "2011-09-04", "2011-08-20", "2010-12-01", "2013-02-01")
                        , col5 = (1:5)/5)
        , col.types = c(col1 = "integer", col2 = "double", col3 = "character", col4 = "date", col5 = "percent")
        , layout.properties = tableProperties(
                        data.text = textProperties( color = "#354B5E" )
                        , data.par = parProperties( text.align = "right" )
                        , data.cell = cellProperties( border.color= "#354B5E" )
        )
)
